# stock-portfolio-tracker
A Python script that tracks your stock portfolio, calculates performance metrics and benchmarks against the S&P 500. 

# What it does
- Tracks multiple stocks using yfinance
- Calculates total return, volatility and Sharpe ratio
- Benchmarks your portfolio against the S&P 500
- Generates a chart of portfolio value over time and allocation breakdown
- Exports results to Excel and saves snapshots to a SQLite database
- 
# Tech Stack
- yfinance — Fetch historical stock data
- pandas — Data manipulation
- numpy — Statistical calculations
- matplotlib — Data visualization
- openpyxl — Excel export
- sqlite3 — Local database storage
